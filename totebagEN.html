WEB CATALOG FOR THE TARTU ART SCHOOL ESTONIAN-JAPANESE-CREE SUSTAINABLE CONSUMPTION TOTEBAG PROJECT

The sustainable cotton totebag project of Tartu Art School was in need of an international web catalog that would introduce the project with clarity about the artwork usage rights when interested in using any of the designs to spread the sustainable message.

It was crucial that the visual identity and logo of Tartu Art School get implemented in the web catalog design.

The web catalog was built in web platform Wix according to the criterias. ​

The web catalog for Tartu Art School in desktop, tab and mobile view.
